# 🧮 Advanced BMI Calculator

A modern and responsive **BMI (Body Mass Index) Calculator** built using **HTML, CSS, and JavaScript**. This project provides an intuitive user interface, animated BMI speedometer gauge, health classification tables, and useful health information to help users understand their BMI results.

---

## 🚀 Features

* ✅ Calculate BMI instantly
* ✅ Support for multiple height units

  * Centimeters (cm)
  * Meters (m)
  * Feet (ft)
  * Inches (in)
* ✅ Support for multiple weight units

  * Kilograms (kg)
  * Pounds (lb)
* ✅ Modern dark-themed UI
* ✅ Responsive design for desktop and mobile
* ✅ Animated speedometer-style BMI gauge
* ✅ BMI classification display
* ✅ Healthy weight range estimation

## 📊 BMI Categories

| Category          | BMI Range   |
| ----------------- | ----------- |
| Severe Thinness   | < 16        |
| Moderate Thinness | 16 - 16.9   |
| Mild Thinness     | 17 - 18.4   |
| Normal Weight     | 18.5 - 24.9 |
| Overweight        | 25 - 29.9   |
| Obesity Class I   | 30 - 34.9   |
| Obesity Class II  | 35 - 39.9   |
| Obesity Class III | ≥ 40        |

---

## 💡 How It Works

1. Enter your age and gender.
2. Enter your height and select the height unit.
3. Enter your weight and select the weight unit.
4. Click **Calculate BMI**.
5. View:

   * BMI Score
   * BMI Category
   * Animated Speedometer Gauge
   * Healthy Weight Range
